## [v1.0] - 2025-11-15

### English
**Added**
- Creation of relational model in **SQL Server** with tables: clients, products, sales, segments, contact_channels, and regions.
- Initial population of the **sales** table with sample data.
- Implementation of **DAX measures** in Power BI:
  - Current Country
  - Current Product
  - Current Year
- Dashboard in **Power BI** with:
  - KPIs in cards
  - Donut, bar, column, and line charts
  - Slicers by time, country, product, and segment
  - Choropleth map by region/country
- Documentation in **README_EN.md** and **README_ES.md** with cross-links.
- Dashboard screenshot (`dashboard-main-page.png`) added in `docs/screenshots/`.

**Notes**
- This version marks the start of the project as a professional portfolio (stable v1.0).
- Future versions will include integration with **MongoDB Atlas** and automation with **Python**.
